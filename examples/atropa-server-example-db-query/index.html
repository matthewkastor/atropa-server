This is an example HTML file for Atropa Server. If you've copied all of the files into your project root, sqliteQuery.jsn should be located here. By specifying various values for the q parameter you'll execute different queries against the database. The buttons will change the contents of the pre tag at the bottom of the page when you've got the server running.

  • /sqliteQuery.jsn?q=table
  • /sqliteQuery.jsn?q=hit_count
  • /sqliteQuery.jsn?q=ip_addresses
  • /sqliteQuery.jsn?q=browsers
  • /sqliteQuery.jsn?q=queries
  • /sqliteQuery.jsn?q=reset
